# Simple Clicker

## Features

* Click-based progression system
* Upgrade mechanics
* Animated UI elements
* Progress scaling
* Clean browser-playable interface

## How to Play

1. Click the "Click me" button to gain clicks.
2. Buy upgrades to increase production.
3. Unlock better progression as your numbers grow.
4. Unlock more stuff to watch your clicks grow.

## Running the Project Locally

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
